# Prueba Técnica QA FullStack - MakersPay

Este repositorio contiene la solución integral a la prueba técnica, demostrando habilidades en automatización, análisis funcional y pruebas de API.

## Módulos del Proyecto
1. **Automatización Web:** Suite de Smoke Tests para SauceDemo (Selenium + Python + POM).
2. **Módulo Funcional:** Análisis de requerimientos y reporte de Bugs para MakersPay.
3. **Módulo API:** Ciclo CRUD completo (POST, GET, PUT, DELETE) con persistencia de datos.

## Notas Técnicas (API)
Se utilizó **MockAPI.io** para las pruebas de API para garantizar la persistencia de datos y la validación de integridad entre peticiones (extraer ID y comparar campos), superando las limitaciones de las APIs de simulación estándar.

## Estructura
- `/automation-saucedemo`: Scripts de Selenium.
- `/functional-makerspay`: Escenarios y Estrategia.
- `/api-reqres`: Colección de Postman y evidencias fotográficas.
